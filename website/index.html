Built for the local stack

One happy path: discover → fit check → deploy → code. Failures are visible, logged, and recoverable.

⌕

HF model discovery

Search Hugging Face, browse popular coding models, inspect quants, licenses, and estimated sizes.

⚡

One-click deploy

Deploy any quantization to Ollama, llama.cpp, vLLM, or SGLang. Oversized? Warn and continue — never hard-block.

▣

GPU-aware

Detect GPUs, predict VRAM fit, and surface free vs total memory before you pull a model.

⌘

Full coding agent

Tools, skills, MCP servers, optional subagents — a real agent loop in a keyboard-first TUI.

▤

Benchmarks

Run suites locally against your model+quant. Publish results when signed in for community data.

☁

Optional cloud

Burst to RunPod, Vast.ai, or Akash. Crypto top-up with USDC on Base for managed Akash deploys.

Install in one line

Scripts install Rust if needed, build from source, and put localcode on your PATH.

curl -fsSL https://raw.githubusercontent.com/view321/LocalCode/main/scripts/install.sh | bash

Requires curl, a C toolchain, and network access. Installs to ~/.local/bin by default.

irm https://raw.githubusercontent.com/view321/LocalCode/main/scripts/install.ps1 | iex

Run in PowerShell. Installs to %USERPROFILE%\.local\bin and offers to add it to your user PATH.

git clone https://github.com/view321/LocalCode.git
cd LocalCode
cargo install --path crates/localcode-cli

Requires Rust 1.75+. Binary lands in ~/.cargo/bin.

After install

# Launch the TUI
localcode

# Diagnostics
localcode doctor --json

# Headless agent
localcode agent run "List files in the workspace" --workspace .
